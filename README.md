# cookkit_php

졸업작품 쿡키트에 사용된 PHP 파일입니다.



그중, 웹 관리자 페이지에 대한 설명입니다.
관리자 페이지의 로그인 화면입니다.

로그인 이후에 상단바에서 유저와 레시피, 공지사항, 밀키트 구매자, 투표 목록을 확인할 수 있고,
공지사항, 밀키트 설명을 작성할 수 있습니다. 작성된 공지사항과 밀키트 설명은 앱에서 확인할 수 있습니다. 부트스트랩을 활용했습니다.
